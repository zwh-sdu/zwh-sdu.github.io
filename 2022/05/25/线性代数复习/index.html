Matrices 矩阵

Inverse 矩阵的逆

  • AA−1=A−1A=IAA^{-1}=A^{-1}A=I AA−1=A−1A=I

  • The inverse A−1A^{-1}A−1 exists if and only if A is ono-singular

∣A∣≠0|A|\neq0 ∣A∣​=0

Pseudo-inverse 伪逆

A+=[ATA]−1ATA^+=[A^TA]^{-1}A^T A+=[ATA]−1AT

  • Assuming ATAA^TAATA is non-singular 假设ATAA^TAATA是非奇异方阵
  • 在 A−1A{-1}A−1 不存在的时候使用，i.e，A不是方阵或 A is singular（奇异矩阵，即|A|=0）

For a square matrix AAA（对称矩阵）

  • Positive definite 正定矩阵（所有特征值大于0）:

    ​ xTAx>0x^TAx > 0xTAx>0 for all x≠0x\neq0x​=0

  • Semi-positive definite 半正定矩阵:

​ xTAx≥0x^TAx\geq0xTAx≥0 for all x≠0x\neq0x​=0

Trace 矩阵的迹

对角线元素的和

  • tr(A)=tr(AT)tr(A)=tr(A^T) tr(A)=tr(AT)

  • Linearity

tr(αA+βB)=αtr(A)+βtr(B)tr(\alpha A+\beta B)=\alpha tr(A)+\beta tr(B) tr(αA+βB)=αtr(A)+βtr(B)

  • Cyclic property

tr(ABC)=tr(BCA)=tr(CAB)tr(ABC)=tr(BCA)=tr(CAB) tr(ABC)=tr(BCA)=tr(CAB)

tr(AB)≠tr(BA)tr(AB)\neq tr(BA) tr(AB)​=tr(BA)

Eigenvectors and Eigenvalues 特征向量和特征值

定义： vvv is an eigenvector of matrix A(mXm) if there exists a scalar λ\lambdaλ ,such that:

Av=λvAv=\lambda v Av=λv

  • v: an eigenvector
  • λ\lambdaλ: the corresponding eigenvalue

理解： an eigenvector represents an invariant direction in the vector space.

特征向量表示在特征空间中不变的方向。

  • 一个被向量 vvv 定义的点方向保持不变。
  • 大小被对应的特征值 λ\lambdaλ 相乘

Similar Matrix 相似矩阵

​ 对于两个 n 阶方阵 A、B 如果存在可逆矩阵 P 使得 B=P−1APB=P^{-1}APB=P−1AP，则 A、B 相似。

  • rank(A)=rank(B)rank(A)=rank(B)rank(A)=rank(B)
  • tr(A)=tr(B)tr(A)=tr(B)tr(A)=tr(B)
  • ∣A∣=∣B∣|A|=|B|∣A∣=∣B∣
  • AAA and BBB have the same eigenvalues

Eigenvalue Decomposition 特征值分解

​ 对于含有 mmm 个线性无关特征向量的方阵 AAA (m*m)，我们可以进行特征值分解

AV=VΛAV=V\Lambda AV=VΛ

A=VΛV−1A=V\Lambda V^{-1} A=VΛV−1

  • Columns of VVV are eigenvectors of AAA
  • Diagonal elements of Λ\LambdaΛ are eigenvalues of AAA

Λ=diag(λ1,λ2,……,λm),λi⩾λi+1\Lambda=diag(\lambda_1,\lambda_2,……,\lambda_m),\lambda_i\geqslant\lambda_{i+1} Λ=diag(λ1​,λ2​,……,λm​),λi​⩾λi+1​

  • 如果 A 是非奇异的，则所有特征值非零

  • if AAA is real and symmetric 实对称方阵：

    • All eigenvalues are real

      ​ if ∣A=λI∣=0|A=\lambda I|=0∣A=λI∣=0 and A=AT⇒λ∈RA=A^T\Rightarrow\lambda\in\RA=AT⇒λ∈R

    • The eigenvectors for distinct eigenvalues are orthogonal 不同特征值对应的特征向量是正交的

  • 如果 A 是正定的，则所有特征值都是正数。

Singular Value Decomposition(SVD) 奇异值分解

定义

Any m×nm\times nm×n matrix AAA of rank rrr can be decomposed into: A=UΣVTA=U\Sigma V^TA=UΣVT

  • for m>nm>nm>n
  • UUU 的每一项都是正交的orthogonal, UUT=Im×mUU^T=I_{m\times m}UUT=Im×m​
  • VVV 的每一项都是正交的orthogonal, VVT=In×nVV^T=I_{n\times n}VVT=In×n​
  • We arrange the singular values as σ1≥σ2≥⋯≥σr>0\sigma_1\geq\sigma_2\geq\cdots\geq\sigma_r>0σ1​≥σ2​≥⋯≥σr​>0
  • UUU 的每一项是 AATAA^TAAT 的特征向量
  • VVV 的每一项是 ATAA^TAATA 的特征向量
  • Eigenvalues λ1,⋯ ,λr\lambda_1,\cdots,\lambda_rλ1​,⋯,λr​ of AATAA^TAAT are the eigenvalues of ATAA^TAATA
  • Singular value 奇异值 σi=λi\sigma_i=\sqrt{\lambda_i}σi​=λi​​
